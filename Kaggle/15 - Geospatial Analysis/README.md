This course covered working with geospatial data and creating interactive maps.

Topics:
- GeoPandas (Python library) and GeoDataFrames
- Map projection and coordinate reference systems (CRS)
- Folium (Python library) and interactive maps (bubble maps, heatmaps, choropleth maps)
- Geocoding (converting a name/address into a locations on a map)
- Table joins for GeoDataFrames (attribute join and spatial join)
- Proximity analysis (measuring distances and finding locations within a radius)
